L'application est compatible PWA et donc elle peut être installée sur ordinateur ou mobile comme une application smartphone.
D'origine l'application utilise directement l'imprimante par défaut du système.
Pour choisir manuellement l'imprimante avant l'impression , il faut renommer le fichier app.js avec le nom que vous voulez et ensuite renommer le fichier app_manuel.js en app.js
